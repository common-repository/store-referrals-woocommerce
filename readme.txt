Store Referrals is a new kind of referral program that lets you spread awareness of your store using tiered rewards.

== Description ==

[Store Referrals](https://www.storereferrals.com) allows you to create, manage, and track multiple campaigns with tiered rewards. This plugin will track all referred traffic and then update your dashboard analytics in real time. Also, you can optionally display a small prompt for customers to join your referral program after checkout.

* Create unlimited campaigns with tiered rewards.
* Set how many clicks and conversions are required to earn each reward.
* Add sample social media posts for your referrers to use when promoting your store.
* Easily recruit your customers to join to your campaigns.
* Export lists of referrers who have qualified for rewards.
* Get real time analytics of how many clicks, conversions, revenue, and referrer sign ups your campaigns have generated.

== Installation ==

1. Install WooCommerce.
1. Install the 'Store Referrals for WooCommerce' plugin through the WordPress plugins screen directly, or upload the plugin files to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Navigate to the Store Referrals settings page (WooCommerce -> Store Referrals).
1. Enter the subscribe URL for one of your campaigns.
